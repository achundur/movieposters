How it came to be

I started drawing a lot of knives because I thought they were fun to draw- it adds an eerie and offputting aura to an image, especially if they aren't expected. I ended up drawing quite a few things that were knife-related or knife-adjacent, so my friends held an intervention, which was very funny. This poster is a response that (sorry guys!).

Design inspiration: American Psycho

The movie poster designs for Amerian Psycho are so so sick (both official and unofficial, check out some of my favs here). I thought that the iconic movie poster (right) was such a perfect concept for this poster, so I ran with it. It was pretty embarrassing to take the reference photos for this poster, so please appreciate the turmoil I went through to make this.

A mid-process shot, and some general vibes.

The "Knifey Materials Production" is a running bit throughout the posters, obviously inspired by this one. It's really funny to me in the way tha dad jokes are funny to dads.

This was the very first poster that I made, so it holds a special place for me. It was also I felt the most challenging one to design. One, because at that point, I hadn't done much visual design in a very long time, so I was dusting off the metaphorical cobwebs. Two, there were a lot of elements I wanted to include (I was playing around with different typography, including a nod to the business cards, and working with an image), and a lot of photo editing required for the main visual element (thanks Photoshop).
